# Java 언어의 6가지 특성
## 1. 운영체제 독립적이다.
- JVM(Java Virtual Machine)이 있어 운영체제(OS) 독립적이다.
- JVM은 OS 독립적이지 않고, OS별 JVM이 설치되어야 한다.
- Java로 작성된 프로그램은 JVM을 거쳐 OS에 전달된다.

## 2. 자동 메모리 관리 기능인 Gabage Collector가 있다.
- 프로그래머가 메모리 관리를 안해줘도 된다.
- 개발자는 비즈니스 로직에만 집중할 수 있어 생산성이 늘어난다고 한다.

## 3. 네트워크, 분산 처리와 관련된 API를 제공한다.자바 언어의 특징.md
- 네트워크 관련 프로그램을 빠르게 작성할 수 있다고 한다.

## 4. 멀티 쓰레드(Multi Thread) 기능을 제공한다.
- 다른 언어는 Mutli Thread 기능을 쓰려면 OS에 따라 구현 방법이 달라진다.
- Java는 OS와 상관없이 구현 가능하다.
- 멀티 스레드 스케쥴링은 자바 인터프리터(Java Interpreter)가 담당한다.

## 5. 동적 로딩(Dynamic Loading)을 지원한다.
- Java 응용 프로그램은 여러 class로 구성되어 있고, 해당 클래스가 필요할 때만 로딩된다.
- 일부 class가 변경되어도 전체 프로그램을 리컴파일 할 필요가 없다.

## 6. 객체 지향 언어다.
- 객체 지향적으로 코딩을 할 수 있도록 언어가 설계되어있다.
